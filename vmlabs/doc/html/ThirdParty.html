Third Party Apps

VM Labs SDK Internal Release 0.86.2
June 14, 2001

Several third party applications are bundled with the SDK. You will have to install these as a separate step from the regular SDK installation. These are:

Application Linux Windows Remarks
tcl/Tk 8.11 included in most Linux installations $(VMLABS)/bundled/windows/tcl811.exe tcl/Tk is required for many SDK tools, such as the Puffin2K debugger.
tcl/Tk 8.32 included in most Linux installations $(VMLABS)/bundled/windows/tcl832.exe tcl/Tk is required for running the Puffin5 debugger.
3d studio max plugin NA $(VMLABS)/bundled/windows/studiomax.zip Unzip this and follow the directions in the readme files.
