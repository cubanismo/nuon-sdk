M3DL Library

The M3DL library has had many updates since the last SDK release.  These include:

  • Ability to draw large images into the frame buffer.  It is no longer necessary to break a large image into smaller images arranged in a grid.  However, please note that this is not the same thing as full-blown support for large textures.  This feature draws only rectangles that are parallel to the viewplane, and would primarily be used for drawing background images prior to rendering other graphics.
  • Now runs properly under MPE 3 and the ROM-based BIOS.   However, it does not yet support the ability to run rendering code on MPE 0 at the same time as the mini-BIOS.  This means that your application must either avoid using MPE 0 for rendering, or it must shut down the mini-BIOS (and give up those services) in order to use MPE 0.
  • Improved documentation -- More detailed information on many items.
  • More samples - New samples include a blending demonstration, and how to mix M3DL graphics with scaled text output from MML2D.

Please see the M3DL library documentation for more information.
