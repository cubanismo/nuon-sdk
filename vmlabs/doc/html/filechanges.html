File Changes

New Files

Deleted Files

 Please note that some of these items may refer to groups of files, since it would be pointless in most cases to individually include every last file.

New Files 

The following files or groups of files have been added to the SDK since the previous release

  • SAMPLE -- There are a lot of new samples.  For a complete overview, please see the Sample Code Overview.
  • LIB\LIBMOVIE.A -- This is the library containing the movie playback functions.
  • LIB\LIBPE.A -- This library contains the interfaces to the ROM entry points for the DVD Presentation Engine.  This is required by the _PlayClip() function.

Deleted Files

The following files from earlier SDK releases are no longer included in the current SDK release, or are now considered obsolete.

  • LIB\LIBBIOS*.A - All variations on the LIBBIOS.A file have been deleted.  Targeting a RAM-based BIOS is no longer appropriate for application development.  If you have special requirements, or you are having difficulty updating an older application to use the ROM-based BIOS, please contact developer support for assistance.
  • INCLUDE\MERLIN\MLEXCEP.H - This file has been deleted.  The appropriate contents have been moved to other header files.
  • BIN\COFF2MPO.EXE - Converted COFF executable files to MPO format.  Required back in the old days when debugging options for COFF files were limited.  No longer needed.
  • BIN\MPO2COFF.EXE - Converted MPO executable object modules to COFF format.   Required back in the old days when output options for COFF files where limited.   No longer needed.
  • BIN\PUFFIN.EXE - Old DOS-based console version of PUFFIN Debugger.  Superceded by newer PUFFIN2K.
  • BIN\PUFFINW.EXE - Old Windows-based version of PUFFIN Debugger.  Superceded by newer PUFFIN2K.
  • BIN\MERLIN.DLL - Functions have been integrated into the main executable files
  • BIN|MMATH.DLL - Functions have been integrated into the main executable files
  • BIN\MDEBUG.DLL - Functions have been integrated into the main executable files
  • BIN\MPACKET.DLL - Functions have been integrated into the main executable files
  • BIN\TRANSFER.MPO - Functions have been integrated into the main executable files
  • \CFG\PUFFINW.INI - INI File used by PUFFINW.EXE.
  • DOC\HTML\LLAMA - The HTML version of the Llama documentation is no longer included.   See the PDF version instead.  Maintaining the same document in both PDF and HTML versions was redundant.
  • DOC\HTML\MML3D - The HTML version of the MML3D library documentation is no longer included.  See the PDF version instead.  Maintaining the same document in both PDF and HTML versions was redundant.
  • DOC\HTML\MUTIL - The HTML version of the MUTIL library documentation is no longer included.  See the PDF version instead.  Maintaining the same document in both PDF and HTML versions was redundant.
