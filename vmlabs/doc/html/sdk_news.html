What's New in this release?

If you have used previous releases of the NUON SDK, please see the information below for an idea of what's changed with the current release.

  • This SDK release is designed to work with a ROM-based BIOS using MPE 3 as the primary processor.  Using a RAM-based BIOS and/or MPE 0 as the primary processor will still work.  Please see Running on ROM BIOS for all the details.
     
  • You must update your system's Flash ROM with the latest ROM-based BIOS code.  (see Installing the ROM-Based BIOS)
  • The stub and MLOAD tool have been updated.  This should improve the reliability of the Ethernet link for the media access functions.  (see Updating Your Stub)
  • Many C / C++ Header Files have been significantly revised.  (see Include Files)
  • You may now configure the address of the program stack.  (See Compiler Changes)
  • See our Library Overview and Sample Code Overview pages.
  • For a list of file additions and deletions, see File Changes
