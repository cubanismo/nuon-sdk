Include File Updates

VM Labs SDK Internal Release 0.86.2
June 14, 2001

The following include file updates are effective with this release:

File

Path

Comments
ioctl.h include/sys Added codes for keyboard drivers.
video.h include/nuon Added _VidSetOutputType
tcp_ip.h include/sys

Added this file.

Enables networking.

dma.h

mutil.h

include/nuon dma.h has been separated from the mutil project. All mutil specific functions have been removed from dma.h and are now in mutil.h.
Protocol.h, objc-api.h, sarray.h, NXConstStr.h, encoding.h, objc-list.h, thr.h, Object.h, hash.h, objc.h, typedstream.h include/objc Removed; Objective C is not supported.

Note that after you install the SDK in your development environment, you must recompile and relink any code that depends on these header files.
