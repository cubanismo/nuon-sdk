This document provides an overview of the sample code included with the NUON SDK that doesn't easily fit into any of the other categories.

The table below shows the folder name of each sample and a brief description of what is demonstrated.  All folder names are relative to the \VMLABS\SAMPLE directory.

Folder Name

Description
Miscellaneous\VMBalls3 How to draw a number of sprite primitives using the BIOS DMA functions.
Miscellaneous\Ycrcb Layout of the YCrCb color space.
