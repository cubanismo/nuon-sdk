This document provides an overview of the Media Access sample code in the NUON SDK.

The table below shows the folder name of each sample and a brief description of what is demonstrated.  All folder names are relative to the SDK's SAMPLE directory.

Folder Name

Description
Media-Access\Deviceinfo How to retrieve information about a media device.
Media-Access\Read-Callback How to use a read callback to determine when a data read has completed.
