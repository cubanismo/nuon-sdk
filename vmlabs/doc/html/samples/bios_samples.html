This document provides an overview of the BIOS sample code in the NUON SDK.

The table below shows the folder name of each sample and a brief description of what is demonstrated.  All folder names are relative to the SDK's SAMPLE directory. 

Folder Name

Description
BIOS-Stuff\LoadGame This demonstrates how to use the system's LoadGame() function to load a new COFF file and execute it.
BIOS-Stuff\TimerTest This program how to use the system's hardware timer.
