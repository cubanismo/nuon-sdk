This document provides an overview of the Games sample code in the NUON SDK. The table below shows the folder name of each sample and a brief description of what is demonstrated. All folder names are relative to the SDK's SAMPLE directory. 

Folder Name

Description

Games\Chomp

Game that shows how to use the sprite library.

Games\Open_SameGame

A simple yet mesmerizing game where you delete groups of tiles for points. The larger the group, the more points you earn! Removing all the tiles on the screen wins you a bonus too!
