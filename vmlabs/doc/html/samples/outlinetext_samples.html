This document provides an overview of the sample code in the NUON SDK that demonstrates the use of outline text fonts that can be scaled to any desired size.

The table below shows the folder name of each sample and a brief description of what is demonstrated.  All folder names are relative to the SDK's SAMPLE directory.

Folder Name

Description
Outline Text\truetype How to use a TrueType scalable font in your application.
Outline Text\txdemo A variety of features associated with outline text.
