This document provides an overview of the sample code in the NUON SDK that demonstrates how to use game controllers.

The table below shows the folder name of each sample and a brief description of what is demonstrated.  All folder names are relative to the SDK's SAMPLE directory. 

Folder Name

Description
Game-Controllers\Controller-Dump This is a diagnostic program that dumps all of the data from each available controller slot.

Slot 0 corresponds to a player's Infrared Remote (IRR) control.  Slots 1 is the first controller found connected to the first controller port.  Slots 2-4 are available when a 4-player adapter is connected.  Slots 5-9 are used for the second controller port.

In addition to basic button and analog controller data, the program also displays whatever additional information is available from the IRR control.

Game-Controllers\deadzone How to implement a center dead-zone, where small analog values are discarded.  This required because most analog joysticks (or other analog controllers) do not provide an exact center position of ZERO in either axis.
