This document provides an overview of several diagnostic sample programs that are included with the NUON SDK.

The table below shows the folder name of each sample and a brief description of what is demonstrated.  All folder names are relative to the SDK's SAMPLE directory.

Folder Name

Description
Diagnostics\Sysinfo Retrieves and displays information about the BIOS version and processor cache settings
Diagnostics\version This sample links in many of the NUON libraries.  When you execute the program, it will display the version number for each library included.
