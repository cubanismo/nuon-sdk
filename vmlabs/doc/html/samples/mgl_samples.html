This document provides an overview of the mGL 3D Graphics Library sample code in the NUON SDK.

The table below shows the folder name of each sample and a brief description of what is demonstrated.  All folder names are relative to the SDK's SAMPLE directory.

Folder Name

Description
mgl\blend How to use the blending features of the mGL 3D library.
mgl\room How to render a scene composed of objects exported using the 3D Studio MAX export plug-in
mgl\simple Creates a simple, non-textured 3D object and animates its orientation and screen position.
