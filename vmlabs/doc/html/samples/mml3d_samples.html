This document provides an overview of the MML3D graphics library sample code in the NUON SDK.

The table below shows the folder name of each sample and a brief description of what is demonstrated.  All folder names are relative to the SDK's SAMPLE directory.

Folder Name

Description
mml3d\cubes Displays a number of 3D cubes.
mml3d\mixed2d Combines 2D graphics output from the MML2D library with 3D graphics from the MML3D library.
mml3d\walkthru A complex sample, showing various features of MML3D.
