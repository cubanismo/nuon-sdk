This document provides an overview of the audio-related sample code in the NUON SDK. The table below shows the folder name of each sample and a brief description of what is demonstrated. All folder names are relative to the SDK's SAMPLE directory. 

Folder Name

Description
audio\MIDI-Custom-Bank Using a custom wavetable data bank with the NUON synth.
audio\MIDI-Parser Bare minimum code (and it doesn't take much) to play a MIDI tune using the NUON synth.
audio\Mixer Using the mixer and panning functions.
audio\PCMPlaySample Usage of the PCMPlaySample function.  It simply plays a short PCM-encoded audio tone.  This is similar to the PCMVoiceOn function, but requires different initialization and offers a different set of options.
audio\PCMPlaySample-SDRAM Basically the same as the PCMPlaySample program, except also demonstrates how to use the tools to specify that a program segment (the PCM data in this case) should be loaded into a particular area of memory.
audio\PCMSetPitch Using the PCMSetPitch function to dynamically change the pitch of a PCM voice.
audio\PCMVoiceOn Use of the PCMVoiceOn function.  It simply plays a short PCM-encoded audio tone.
audio\PlayWav How to play WAV and AIFF format audio.
