Document formats

Each document is provided in one of more of the following formats.

  • Adobe Acrobat (.pdf)

  • Ascii text

  • HTML

  • MS Help

Note that effective with this release of the SDK, MS Word format is no longer used for documentation.

Viewing the Documents.

HTML documents are kept simple and may be viewed with the web browser of your choice.

Ascii text documents are intended to be readable on both Windows and Linux environments.

Adobe Acrobat documents require Acrobat Reader 4.0 from Adobe. You can get this in one of two ways:

  • Contact the VM Labs system administrator to have Adobe Acrobat installed on your machine. (Windows users only)

Or

  • Download it from the Adobe website.

Note: Internet Explorer may have a problem displaying PDF files over the web server. A workaround is to disable to the Adobe Acrobat plug-in by doing the following:

  1. Exit from Internet Explorer.
  2. Start the Acrobat 4.0x viewer.
  3. Choose File > Preferences > General.
  4. Deselect Web Browser Integration in the Options section and then click OK.
  5. Exit from the Acrobat 4.0x viewer.
  6. Restart Internet Explorer. The next time you select a link to a PDF file in Internet Explorer, a dialog box will prompt you to specify what to do with the file. If you select "Open this file from its current location," Internet Explorer opens the PDF file in the Acrobat viewer as a helper appliction. If you select "Save this file to disk," Internet Explorer saves the PDF file to your hard disk, from where you can open it later.
