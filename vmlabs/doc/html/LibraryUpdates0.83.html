Library Updates

VM Labs SDK Release 0.831
December 1, 2000

The following library updates with this release.

File

Comments
libmml2d.a

Version 2.21

  • Now allows 32 to 16 bit conversion in movepixdirect.
  • Added colorSafe functionality (see Chapter 5 of the mml2d documentation.). Uses sqrt function from libm.a.
libmltxt.a

Version 2.4

  • Now correctly puts the glyphs exactly on the baseline.
  • Improved the correspondence between mmltxt and Impulse.
libmgl.a

Version .83

Stabilized code. See mgl release notes.

libbios.a, librombios.a, libHALs.a, libpe.a

  • No longer corrupts the register r15. This register is used by the level 1 ISR to save the rc1 counter.
  • Includes the fixes for the problem with reading from the second layer of a multilayer disk..
  • Includes the DCache configuration change to 2k 2-way
libc.a, libc_p.a Fixes problem of opening many files.
crt0.o, crtend.o, crt0l.o, gcrt0.o, gcrt0l.o Includes the patch that forces the comm section to exist in crt0.o.

After installing the new SDK on your system, you must re-link your NUON applications.
