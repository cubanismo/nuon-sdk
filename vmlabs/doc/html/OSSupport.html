OS Support

VM Labs SDK Internal Release 0.86.2
June 14, 2001

The SDK runs on a number of flavors of Windows and Linux, but is officially supported on a more limited subset. The following table indicates the level of support for each.

These levels of support have the following meanings:

Support Level What it means
Official The SDK is tested on this OS and is expected to function correctly in all respects.
Runs The SDK has been partially tested on this OS, and may reasonably be expected to function normally in most respects. However, some functionality may not be available, or there may be bugs that are not present on the officially supported platform. At VM Labs' discretion, VM Labs may fix such problems, or you may be requred to upgrade your OS if the desired functionality is available on an officially supported release.
Not Supported The SDK has not been tested at all on this OS. It may or may not function correctly. If you are experiencing problems, you must upgrade to an officially supported version.

Windows 95

Starting with SDK 0.82, any official support for the VM Labs SDK is withdrawn for Windows 95. If you are running Windows 95 and experience problems, you must upgrade your OS.

Windows ME

This OS release is too new to be considered. Future releases of the SDK will be supported at the "Runs" level on Windows ME. The officially supported MicroSoft Windows platform will be Windows 2000.

Future Support

Future release of the VM Labs SDK will be migrating to newer OS versions. In particular, the following OS versions are targeted:

    • Windows 2000
    • Linux RedHat 7.0
    • Linux SUSE 7.2

Migration of support to these platforms is expected to begin with the next release of the SDK.
