Overview

This SDK contains a variety of code samples. The majority of the samples can be compiled with the tools in this SDK, and executed on your NUON DVD player.  A few samples are incomplete in some way and are provided merely so you can examine the source code.

All of the sample code in the NUON are located in the SDK's SAMPLE directory.  Inside that directory are a number of subdirectories; one for each category of samples.  The individual code samples are subdirectories within the category directories. Each sample demonstrates a particular concept, such as how to use a certain NUON library, or how to use the SDK tools in a particular way.

Notes about MAKEFILEs for samples

The MAKEFILEs use a special include file named NUON_BUILD_TOOLS.MK.  This file is located in the SDK's UTIL directory and contains macro definitions and build rules that are required by virtually all NUON software projects.

The NUON_BUILD_TOOLS.MK file is used by all of the samples, but you do not have to use it for your own projects.  If you do use it, please be aware that the contents may change in future releases of the SDK.
