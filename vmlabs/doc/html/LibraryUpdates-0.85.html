Library Updates

VM Labs SDK Internal Release 0.85
March 12, 2001

The following library updates with this release.

File

Comments
libmml2d.a

Version 2.24

  • Extends m2dReadPixels and m2dWritePixels to 8 bit pixels
  • Updated documentation.

NOTE: This library is now dependent on libmutil.a; you must include -lmutil and -lm after -lmml2d when linking.

librombios.a

Version 1.03.16.00

  • New BIOS API _LoadDefaultSystemSettings which restore the system settings to default.
  • Updated documentation

newlib version 1.7.0 nuon version 11

  • Bug fix to memmove

  • Updated documentation
