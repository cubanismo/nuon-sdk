Don't Panic...

If you are just getting your start with the NUON Development System and Software Development Kit (SDK), the first place you should start reading is a document called The Hitchhiker's Guide To NUON.  This document is provided with the SDK in Adobe Acrobat PDF format.  To view or print this document, you must have either Adobe Acrobat or another PDF-viewer installed on your computer.

 

SDK Installation

The Hitchhiker's Guide includes information on how to configure your development system, how to install and configure the SDK programming tools, a mini-reference that covers many of the SDK tools, and much more.

You should read the Flash ROM Update Instructions and Debug Card Stub Update Instructions after each new SDK release to make sure your development system's firmware and stub are up to date.

The best place to go next is the HTML documentation provided with the SDK.  Starting with the file you're reading now, you can link to information about the NUON system's capabilities, programming libraries, development tools, and more.
