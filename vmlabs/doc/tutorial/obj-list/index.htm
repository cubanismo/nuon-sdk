Running the Examples

The associated code for these examples is in the folder "oldemo". The main source file with which you'll be dealing is called "ol_demo2.s". By setting up this file correctly, then saving it and re-assembling the code, you will be able to run the examples. The batch file "m" will reassemble and run the code.
