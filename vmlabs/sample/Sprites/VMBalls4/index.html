This is the 4th major revision example of ball sprites flying around the TV screen.  This version has been revised to use the new LIBSPRITE library, taking advantage of this library's ability to scale and rotate the sprite image.

The program generates a display list of sprites.  It uses DMA to display a background image that contains a stylized version of the VM Labs logo, and then draws the display list of sprites on top of that.  The sprites move around the screen, rotating at different speeds, and increasing and decreasing in size.

The ball sprite data is 32 bits, and the alpha channel is used to determine which portions of the sprite are transparent.

Filename Description
graphics.c A variety of graphics routines.
makefile MAKEFILE for building and running sample program.
proto.h Function prototypes used by program.
sprite.s Assembly language module that does binary include of image data.
vball.raw Raw RGB image file (32*32) for ball sprite.  Converted into YCRCB format when program build is done.
vmballs.c Main program code for sample.
vmballs.h Contains a variety of definitions used throughout the program
vmlogo.raw Raw RGB image file (360*240) for background image.  Converted into YCRCB format when program build is done.
