This program demonstrates how to initialize the system timer and then retrieve the current timer value.  This is then printed to the screen.

This program uses the NUON outline text capabilities for the text that is shown.  It also scales a fairly low-resolution frame buffer to fit the screen and uses the four-tap vertical video filter to keep things looking nice.

Filename Description
timertest.c Main program code for sample.
makefile MAKEFILE for building and running sample program.
