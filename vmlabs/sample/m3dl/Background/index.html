This program demonstrates how to draw large background images using the M3DL library.

Filename Description
bground.c Main source file for sample
bground.h Contains a variety of definitions used throughout the program
datafile.s Assembly file that includes background images as binary data.
makefile MAKEFILE for building and running sample program.
DATA folder containing the graphics images used in the program

This sample requires M3DL version v3.02 (March 29, 2000) or later.
